## Setup your development

Learn how to create and deploy your first Pebbling agent locally.

### Prerequisites

<AccordionGroup>
  <Accordion icon="python" title="Install Python dependencies">
    Install the required Python packages for Pebbling:
    ```bash
    uv venv --python 3.12.9
    source .venv/bin/activate
    uv add pebbling agno openai
    ```
  </Accordion>
  <Accordion icon="gear" title="Set up OpenAI API key">
    Set your OpenAI API key as an environment variable:
    ```bash
    export OPENAI_API_KEY="your-api-key-here"
    ```
  </Accordion>
</AccordionGroup>

### Create your first agent

<AccordionGroup>
  <Accordion icon="file-code" title="Create agent configuration">
    Create a `simple_agent_config.json` file with your agent configuration:
    ```json
    {
      "author": "Your Name",
      "name": "Simple Agent",
      "description": "A basic AI agent for simple question answering",
      "version": "1.0.0",
      "recreate_keys": true,
      "skills": [
        {
          "name": "question-answering",
          "description": "Ability to answer basic questions",
          "version": "1.0.0"
        }
      ],
      "capabilities": {
        "reasoning": true,
        "code_execution": false,
        "web_search": false,
        "file_operations": false,
        "multimodal": false
      },
      "agent_trust": "low",
      "kind": "agent",
      "debug_mode": true,
      "debug_level": 2,
      "monitoring": false,
      "telemetry": false,
      "num_history_sessions": 5,
      "deployment": {
        "url": "http://localhost:8030",
        "expose": true,
        "protocol_version": "1.0.0",
        "proxy_urls": ["https://simple.pebbling.ai"],
        "cors_origins": ["*"]
      },
      "storage": {
        "type": "memory",
        "connection_string": "postgresql+asyncpg://user@localhost:5432/pebble-example"
      },
      "scheduler": {
        "type": "memory"
      }
    }
    ```
  </Accordion>
  <Accordion icon="code" title="Create your agent">
    Create a Python file with your pebblified agent:
    ```python
    import json
    import os
    from typing import List

    from agno.agent import Agent
    from agno.models.openai import OpenAIChat

    from pebbling.common.models import DeploymentConfig, SchedulerConfig, StorageConfig
    from pebbling.common.protocol.types import AgentCapabilities, AgentSkill
    from pebbling.penguin.pebblify import pebblify


    def load_config(config_path: str):
        """Load configuration from JSON with defaults."""
        current_dir = os.path.dirname(os.path.abspath(__file__))
        full_path = os.path.join(current_dir, config_path)
        
        with open(full_path, 'r') as f:
            config = json.load(f)
            return config

    # Load configuration
    simple_config = load_config("simple_agent_config.json")

    @pebblify(
        author=simple_config["author"],
        name=simple_config.get("name"),
        description=simple_config.get("description"),
        version=simple_config.get("version", "1.0.0"),
        recreate_keys=simple_config.get("recreate_keys", True),
        skills=[AgentSkill(**skill) for skill in simple_config.get("skills", [])],
        capabilities=AgentCapabilities(**simple_config["capabilities"]),
        agent_trust=simple_config.get("agent_trust"),
        kind=simple_config.get("kind", "agent"),
        debug_mode=simple_config.get("debug_mode", False),
        debug_level=simple_config.get("debug_level", 1),
        monitoring=simple_config.get("monitoring", False),
        telemetry=simple_config.get("telemetry", True),
        num_history_sessions=simple_config.get("num_history_sessions", 10),
        documentation_url=simple_config.get("documentation_url"),
        extra_metadata=simple_config.get("extra_metadata", {}),
        deployment_config=DeploymentConfig(**simple_config["deployment"]),
        storage_config=StorageConfig(**simple_config["storage"]),
        scheduler_config=SchedulerConfig(**simple_config["scheduler"]),
    )
    def simple_agent(messages: List[str]) -> str:
        """Regular function example - returns single result."""
        
        # Use any framework internally
        agent = Agent(
            instructions="Provide a simple response to the user's message",
            model=OpenAIChat(id="gpt-4o")
        )
        
        result = agent.run(input=messages)
        
        return result.to_dict()['content']
    ```
  </Accordion>
</AccordionGroup>

### Run your agent

<AccordionGroup>
  <Accordion icon="play" title="Start your agent">
    Run your Python file to start your pebblified agent:
    ```bash
    python your_agent.py
    ```
    Your agent will be available at `http://localhost:8030`
  </Accordion>
  <Accordion icon="test-tube" title="Test your agent">
    Test your agent by sending a POST request:
    ```bash
    curl -X POST http://localhost:8030/chat \
      -H "Content-Type: application/json" \
      -d '{"messages": ["Hello, how are you?"]}'
    ```
  </Accordion>
</AccordionGroup>
