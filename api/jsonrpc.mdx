# JSON-RPC Methods

All agent communication uses JSON-RPC 2.0 protocol over HTTP POST to the root endpoint `/`.

## Endpoint

```
POST http://localhost:8030/
Content-Type: application/json
```

---

## message/send

Send a message to the agent and receive a response.

### Request

```json
{
  "jsonrpc": "2.0",
  "method": "message/send",
  "params": {
    "message": {
      "role": "user",
      "parts": [
        {
          "kind": "text",
          "text": "provide sunset quote"
        }
      ],
      "kind": "message",
      "messageId": "msg-123",
      "contextId": "ctx-123",
      "taskId": "task-123"
    },
    "configuration": {
      "acceptedOutputModes": ["application/json"]
    }
  },
  "id": "req-123"
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `message` | object | Yes | Message object |
| `message.role` | string | Yes | `"user"` or `"agent"` |
| `message.parts` | array | Yes | Message content parts |
| `message.kind` | string | Yes | Always `"message"` |
| `message.messageId` | string | Yes | Unique message ID (UUID) |
| `message.contextId` | string | Yes | Conversation context ID (UUID) |
| `message.taskId` | string | Yes | Task ID (UUID) |
| `message.referenceTaskIds` | array | No | Previous task IDs |
| `message.metadata` | object | No | Additional metadata |
| `configuration` | object | No | Request configuration |

### Response

```json
{
  "jsonrpc": "2.0",
  "result": {
    "task": {
      "taskId": "task-123",
      "contextId": "ctx-123",
      "status": {
        "state": "completed",
        "timestamp": "2025-10-26T14:47:52Z"
      },
      "artifacts": [
        {
          "kind": "text",
          "text": "The sunset paints the sky with hope for tomorrow."
        }
      ]
    }
  },
  "id": "req-123"
}
```

## tasks/get

Get the status and details of a specific task.

### Request

```json
{
  "jsonrpc": "2.0",
  "method": "tasks/get",
  "params": {
    "taskId": "task-123"
  },
  "id": "req-124"
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `taskId` | string | Yes | Task ID (UUID) |

### Response

```json
{
  "jsonrpc": "2.0",
  "result": {
    "task": {
      "taskId": "task-123",
      "contextId": "ctx-123",
      "kind": "task",
      "status": {
        "state": "completed",
        "timestamp": "2025-10-26T14:47:52Z"
      },
      "artifacts": [...],
      "history": [...]
    }
  },
  "id": "req-124"
}
```

## tasks/list

List all tasks with optional pagination.

### Request

```json
{
  "jsonrpc": "2.0",
  "method": "tasks/list",
  "params": {
    "limit": 10,
    "offset": 0
  },
  "id": "req-125"
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `limit` | integer | No | Max results (1-100, default: 10) |
| `offset` | integer | No | Offset for pagination (default: 0) |

### Response

```json
{
  "jsonrpc": "2.0",
  "result": {
    "tasks": [
      {
        "taskId": "task-123",
        "status": {...}
      }
    ],
    "total": 42
  },
  "id": "req-125"
}
```

## tasks/cancel

Cancel a running task.

### Request

```json
{
  "jsonrpc": "2.0",
  "method": "tasks/cancel",
  "params": {
    "taskId": "task-123"
  },
  "id": "req-126"
}
```

### Response

```json
{
  "jsonrpc": "2.0",
  "result": {
    "success": true,
    "taskId": "task-123"
  },
  "id": "req-126"
}
```

## tasks/feedback

Submit feedback for a completed task.

### Request

```json
{
  "jsonrpc": "2.0",
  "method": "tasks/feedback",
  "params": {
    "taskId": "task-123",
    "feedback": "Great response!",
    "rating": 5,
    "metadata": {
      "category": "quality",
      "helpful": true
    }
  },
  "id": "req-127"
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `taskId` | string | Yes | Task ID (UUID) |
| `feedback` | string | Yes | Feedback text |
| `rating` | integer | No | Rating 1-5 |
| `metadata` | object | No | Additional metadata |

## Task States

| State | Description |
|-------|-------------|
| `submitted` | Task received, not yet started |
| `working` | Agent is processing |
| `input-required` | Waiting for user input |
| `auth-required` | Waiting for authentication |
| `completed` | Task finished successfully |
| `canceled` | Task was canceled |
| `failed` | Task failed with error |

## Message Parts

### Text Part

```json
{
  "kind": "text",
  "text": "Hello, agent!"
}
```

### File Part

```json
{
  "kind": "file",
  "file": {
    "url": "https://example.com/file.pdf",
    "mimeType": "application/pdf",
    "name": "document.pdf"
  }
}
```

### Data Part

```json
{
  "kind": "data",
  "data": {
    "key": "value",
    "nested": {...}
  }
}
```

## Examples

### With Payment (x402)

```json
{
  "jsonrpc": "2.0",
  "method": "message/send",
  "params": {
    "message": {
      "role": "user",
      "parts": [{"kind": "text", "text": "Hello"}],
      "kind": "message",
      "messageId": "msg-123",
      "contextId": "ctx-123",
      "taskId": "task-123",
      "metadata": {
        "x402.payment.status": "payment-submitted",
        "x402.payment.payload": {
          "resource": "/agent/my-agent",
          "scheme": "exact",
          "network": "base-sepolia",
          "asset": "0x036CbD53842c5426634e7929541eC2318f3dCF7e",
          "payTo": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0",
          "amount": "10000",
          "signature": "0x...",
          "timestamp": "2025-10-26T14:47:52Z",
          "payer": "0x..."
        }
      }
    }
  },
  "id": "req-123"
}
```

### With Task Reference

```json
{
  "jsonrpc": "2.0",
  "method": "message/send",
  "params": {
    "message": {
      "role": "user",
      "parts": [{"kind": "text", "text": "make it shorter"}],
      "kind": "message",
      "messageId": "msg-124",
      "contextId": "ctx-123",
      "taskId": "task-124",
      "referenceTaskIds": ["task-123"]
    }
  },
  "id": "req-124"
}
```
