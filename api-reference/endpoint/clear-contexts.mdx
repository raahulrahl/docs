description: 'Clear a context and all associated tasks'

## Clear Contexts

Clears a specific context or all contexts from the system, removing all associated tasks.

### JSON-RPC Request

```json
{
  "jsonrpc": "2.0",
  "method": "clear_context",
  "params": {
    "context_id": "context-uuid-here"
  },
  "id": "unique-request-id"
}
```

### Parameters

- **context_id** (string, optional): The UUID of the specific context to clear. If omitted, all contexts will be cleared.

### Response

```json
{
  "jsonrpc": "2.0",
  "id": "unique-request-id",
  "result": {
    "message": "All tasks and contexts cleared successfully"
  }
}
```

### Notes

- This operation is irreversible
- All tasks associated with the cleared context(s) will also be removed
- Use with caution in production environments
