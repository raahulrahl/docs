## List Contexts

Retrieves a list of all contexts in the system with optional length limit.

### JSON-RPC Request

```json
{
  "jsonrpc": "2.0",
  "method": "list_contexts",
  "params": {
    "length": 50
  },
  "id": "unique-request-id"
}
```

### Parameters

- **length** (integer, optional): Maximum number of contexts to return

### Response

```json
{
  "jsonrpc": "2.0",
  "id": "unique-request-id",
  "result": [
    {
      "context_id": "context-uuid-1",
      "created_at": "2024-01-01T00:00:00Z",
      "task_count": 5
    },
    {
      "context_id": "context-uuid-2",
      "created_at": "2024-01-01T00:05:00Z",
      "task_count": 3
    }
  ]
}
```

### Error Response

```json
{
  "jsonrpc": "2.0",
  "id": "unique-request-id",
  "error": {
    "code": -32001,
    "message": "No contexts found"
  }
}
```
