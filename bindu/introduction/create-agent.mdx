## Running Your Agent

Navigate to your agent directory and install dependencies:

```bash
cd your-agent
uv sync
```

Start your agent server:

```bash
uv run python -m your_agent.main
```

Your agent is now live at `http://localhost:3773` and ready to communicate using:
- **A2A** - Agent-to-Agent protocol
- **AP2** - Agentic Protocol 2
- **X402** - Payment rails

## Troubleshooting

<AccordionGroup>
  <Accordion title="uv not found">
    Install uv using:
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
    Then restart your terminal.
  </Accordion>
  
  <Accordion title="Port already in use">
    Change the `deployment_port` in your `agent_config.json` or set a different port when prompted during setup.
  </Accordion>
  
  <Accordion title="Authentication errors">
    Make sure you've configured your Auth0 credentials in the environment variables. Check the generated README for details.
  </Accordion>
  
  <Accordion title="Dependencies not installing">
    Try removing the `.venv` directory and running `uv sync` again:
    ```bash
    rm -rf .venv
    uv sync
    ```
  </Accordion>
</AccordionGroup>
