### Using uv (Recommended)

```bash
uv add bindu
```

### Using pip

<Steps>
  <Step title="Create a virtual environment">
    <CodeGroup>
      ```bash Mac
      python3 -m venv ~/.venvs/bindu
      source ~/.venvs/bindu/bin/activate
      ```

      ```bash Windows
      python3 -m venv binduenv
      binduenv/scripts/activate
      ```
    </CodeGroup>
  </Step>

  <Step title="Install bindu">
    <CodeGroup>
      ```bash Mac
      pip install -U bindu
      ```

      ```bash Windows
      pip install -U bindu
      ```
    </CodeGroup>
  </Step>
</Steps>

<Note>
  If you encounter errors, try updating pip: `python -m pip install --upgrade pip` 
</Note>

## Contributing

Want to contribute to Bindu? Here's how to set up your development environment:

```bash
# Clone the repository
git clone https://github.com/getbindu/Bindu.git
cd Bindu

# Install development dependencies with uv
uv sync

# Install pre-commit hooks
pre-commit run --all-files
```

See our [Contributing Guidelines](https://github.com/getbindu/Bindu/blob/main/.github/contributing.md) for more details.
